Rifki Store - PHP Native (Ready-to-run)

Installation:
1. Copy the folder `rifki_store` into your web server root (e.g., C:\xampp\htdocs\).
2. Import `database.sql` using phpMyAdmin.
3. Edit `config.php` if DB credentials differ.
4. Ensure `admin/upload/` is writable.
5. Open: http://localhost/rifki_store/login.php
